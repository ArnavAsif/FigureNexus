> It's a **single-page gaming lobby** вҖ” one immersive environment where every interaction (shop, story, limited edition, product matching) opens as a modal overlay **without ever leaving or reloading the page.**  
> *Built like a game. Runs like a game.*

## вҡЎ The Core Concept вҖ” Why This Is Different

Most Shopify stores navigate between pages. BOOYAH! doesn't.

## вҡҷпёҸ How The Modal System Works

```javascript
// Every nav item triggers a modal вҖ” never a page load
document.querySelectorAll('[data-modal-trigger]').forEach(trigger => {
  trigger.addEventListener('click', (e) => {
    e.preventDefault()                          // вҶҗ no navigation
    const target = trigger.dataset.modalTrigger
    openModal(target)                           // вҶҗ open overlay instead
  })
})

// Modals stack вҖ” product modal opens inside shop modal
function openModal(id) {
  document.getElementById(`modal-${id}`).classList.add('active')
  document.body.classList.add('modal-open')
}

function closeModal(id) {
  document.getElementById(`modal-${id}`).classList.remove('active')
}
```

**Result:** Zero page reloads. The lobby background always stays alive underneath every overlay. The immersion never breaks.

## рҹҡҖ Getting Started

```bash

# 2. Install Shopify CLI
```
